# CS348 Project - Stage 1

## Stack
- Frontend: Next.js
- Backend: Flask

## How to Run

### Backend
1. Navigate to `backend` directory.
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python app.py`

### Frontend
1. Navigate to `frontend` directory.
2. Install dependencies: `npm install`
3. Run: `npm run dev`
